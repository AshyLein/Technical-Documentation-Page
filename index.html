Getting started with CSS

CSS (Cascading Style Sheets) is a styling language used for changing the look and formatting of a document written in HTML. It allows web developers to control the presentation of a webpage, including colors, fonts, layouts, and more.

Syntax

CSS consists of a set of rules, each containing a selector and a declaration block. The selector specifies which HTML elements the styling should be applied to, and the declaration block contains the styling properties and their values. The basic syntax is as follows:


    selector {
      property: value;
    }
Selectors

CSS selectors are used to match elements in an HTML document.
Commonly used selectors:

  • Element selector: targets all elements of a specific type (e.g., h1, p).
  • ID selector: targets an element with a specific ID attribute value (e.g., #myElement).
  • Class selector: targets elements with a specific class attribute value (e.g., .myClass).

    h1 {
      property: value;
    }

    #h1 {
      property: value;
    }

    .h1 {
      property: value;
    }

Properties

CSS properties define the styling characteristics of an element or the entire page layout. Some common properties include:

  • Color: specifies the text color.
  • Font-family: sets the font for the text.
  • Width: sets the width of an element.
  • Display: controls the display behavior of an element.

    selector {
      color: value;
    }

    selector {
      font-family: value;
    }
Values

CSS property values define the specific styles that should be applied. Examples of values include:

  • Color values: hexadecimal, RGB, or color names.
  • Font-family values: specific font names or generic font families.
  • Length values: pixel, percentage, em, etc.

    selector {
      color: rgb(255,255, 255);
    }

    selector {
      height: 2px;
    }
Box Properties

In the CSS box model, every HTML element is treated as a box with four adjustable areas:

  1. content (blue): where content appears (text and images)
  2. padding: area around the content (transparent
  3. border: border that goes around the padding and content
  4. margin: area outside the border around the element (transparent)

Flexbox

Flexbox is a CSS module that provides a more efficient way to arrange and align elements within a container. It introduces a flexible box layout model that allows elements to automatically adjust their size and position based on available space and specified rules. With flexbox, you can create responsive and dynamic layouts without the need for complex calculations or multiple Media queries. It is particularly useful for building navigation menus, grids, and flexible containers. The key concepts of flexbox include:

  • Flex Container: The parent element that contains flex items. It is defined by setting the display property to flex or inline-flex.
  • Flex Items: The child elements within a flex container. They can be manipulated and arranged using various flex properties.
  • Flex Direction: Determines the direction in which flex items are laid out within the flex container. It can be set to row (default), row-reverse, column, or column-reverse.
  • Justify Content: Controls the alignment of flex items along the main axis of the flex container. It can be used to distribute space evenly or align items to the start, end, or center of the container.
  • Align Items: Controls the alignment of flex items along the cross axis of the flex container. It can be used to align items to the top, bottom, center, baseline, or stretch them to fill the container.
  • Flex Wrap: Specifies whether flex items should wrap onto multiple lines or stay on a single line. It can be set to nowrap (default), wrap, or wrap-reverse.
Overall, flexbox simplifies the process of creating flexible and responsive layouts, making it a powerful tool for modern web design.

Media Queries

Media queries in CSS are used to apply different styles based on the characteristics of the device (height. width, resolution) or browser being used to view the webpage. This allows the webpage to be rendered differently according to different screen sizes, resolutions, orientations, and other properties.


    /* Styles for screens with max-width of 600px */
@media (max-width: 600px) {
  body {
    font-size: 14px;
  }
}

/* Styles for screens with min-width of 1200px */
@media (min-width: 1200px) {
  body {
    font-size: 18px;
  }
}

/* Styles for screens between 601px and 1199px */
@media (min-width: 601px) and (max-width: 1199px) {
  body {
    font-size: 16px;
  }
}
